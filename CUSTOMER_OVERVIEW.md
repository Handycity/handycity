# Kundenübersicht: Was Sie mit der Homepage bekommen haben

## Ihr Ergebnis auf einen Blick

Sie haben eine vollständige, moderne Handycity-Homepage mit:
- klarer Service-Navigation
- interaktivem Reparatur-Preisrechner
- Geräte-Retter-Prämie Bereich
- Willhaben-Angebotsbereich
- Kontakt, Standort, Öffnungszeiten
- FAQ, Bewertungen und rechtliche Seiten

## Was Ihre Kunden auf der Seite tun können

- Schnell zum passenden Anliegen springen (Reparieren, Kaufen, Extra-Service)
- Richtpreise für Reparaturen direkt im Rechner sehen
- Sofort anrufen oder Reparaturanfrage starten
- Geräte-Retter-Prämie Infos ansehen
- Aktuelle Geräteangebote über Willhaben öffnen
- Bewertungen lesen
- Standort und Route in Google Maps öffnen

## Was Sie als Betreiber ohne Code ändern können

Über GitHub Actions Formulare:
- Firmeninfos und Hero-Texte
- Öffnungszeiten
- Preisrechner-Einträge
- Service-Einträge
- Willhaben-/Pickup-Texte

Zusätzlich automatisiert:
- Willhaben-Angebote synchronisieren
- Google-Reviews-Backup synchronisieren

## Wie Änderungen live gehen

- Bei Push nach `main` wird automatisch gebaut und auf GitHub Pages deployed.
- Zusätzlich laufen Validierungen (Content, Assets, Build), damit fehlerhafte Inhalte früh erkannt werden.

- Bedienanleitungen: `OWNER_GUIDE.md`, `EDITING.md`
- Diese Kundenübersicht: `CUSTOMER_OVERVIEW.md`
